
# さっかのあめ-執筆支援ツール-

https://wri-sup.pages.dev/

### このツールの目的

ゲーム、長編小説、アニメ...あらゆる創作物において、「エタり」「設定の破綻」は最大の敵です
「さっかのあめ」はそれらを防ぎ、あなたの創作物を一つレベルアップさせます

![alt text](<スクリーンショット 2026-05-27 20.44.32.png>)

##  主な機能 -着地点を明確に、完結だけはできるように-
- 複数の作品を登録・管理
- 作品ごとに「ゴール」「テーマ」を設定
-- ゴールさえ決めておけば、最悪の場合そこに着地させられます

### シーン管理
- シーンの追加・編集・削除
- 上下ボタンによる並び替え
- 作品世界の日時(自由記述)で時系列を管理
- シーン終了時点の世界状態をメモ
- [TODO] 用の置き場フィールド
- 全シーンをテキストとしてクリップボードにコピー(AIに誤字チェックさせるなどに使ってください)

### 伏線管理　-設定の破綻に怯える必要はありません-
- 伏線の追加・編集・削除
- 3ステータス(構想中・張り済み・回収済み)による管理
- ステータスフィルタタブで未回収伏線を絞り込み
- 各伏線を「張ったシーン」「回収シーン」に紐付け
